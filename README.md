## Минусы

## Плюсы:
 - Никогда не будет ловить на парковке
 - Еще не написан


## Сборка

Для отладки используется:
```bash
npm run dev
```

Для сборки и запуска используется:
```bash
npm run start
```

Отдельно сборка:
```bash
npm run build
```
