# Messenger Min

## Минусы
 - Только в названии

## Плюсы:
 - Никогда не будет ловить на парковке
 - Еще не написан

## Changelog:
* Теперь осуществляется валидация пользовательского ввода. В случае ошибок показывается тултип с описанием ошибки
* Теперь если пользовательский ввод успешно проверен, данные не исчезают, а логируются в консоли
* Добавлен класс для работы с запросами, но он пока не используется
* Сам код переписан на TypeScript

## Сборка

Для отладки используется:
```bash
npm run dev
```

Для сборки и запуска используется:
```bash
npm run start
```

Отдельно сборка:
```bash
npm run build
```
